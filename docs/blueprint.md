# **App Name**: CreaSocial

## Core Features:

- User Authentication: Allow users to register, log in, and manage their accounts using Firebase Authentication.
- Short Video Upload and Display: Enable users to upload short videos (shorts) and display them in a TikTok-like vertical scrollable feed. Storage on Firebase.
- Post Creation and Feed: Allow users to create and share text and image posts in a Facebook-like feed. Persisted with Firestore.
- Centralized Content Creation: Implement a central '+' button that allows users to choose between uploading a short video or creating a post.
- User Profiles: Create user profiles to display user-generated content and basic information.
- Basic Interactions: Implement basic like and comment functionality for both shorts and posts using Firestore to keep track of interactions.
- Settings: Implement settings page.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey a sense of modernity and connection.
- Background color: A very light blue (#F0F8FF) to keep a clean and fresh feel.
- Accent color: A bright pink (#FF69B4) to add a pop of fun and highlight interactive elements.
- Body and headline font: 'PT Sans' (sans-serif) will be used for both headlines and body text, since it has both a modern look and some warmth, making it suitable for a social app.
- Note: currently only Google Fonts are supported.
- Use a set of modern, outline-style icons for navigation and actions, ensuring they are intuitive and visually consistent.
- Maintain a simple and modern layout with a bottom navigation bar for easy access to key sections: Profile, Shorts, Post creation, Posts feed, and Settings.
- Incorporate smooth transitions and subtle animations to enhance user experience, such as video loading animations and like/comment interactions.
